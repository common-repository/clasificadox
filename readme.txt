== Description ==

El widget de ClasificadoX añade en tu barra lateral tus anuncios insertados en ClasificadoX

Te permite configurar los siguientes datos:

* Número de anuncios a mostrar
* Tu correo electrónico, para mostrar los anuncios que hayas insertado
* Si no indicas ningún correo, mostraremos anuncios aleatorios
* No necesaria ninguna API key

Demo: http://blog.clasificadox.com/

Description in english:

The ClasificadoX widget adds to your sidebar your advertisements in ClasificadoX

It allows you to configure the following:

* Number of ads to display
* Your email address, to display the ads you've inserted
* If you do not enter any email, show random ads
* No need for any API key

Demo: http://blog.clasificadox.com/

== Installation ==
Descarga y sube el plugin a tu carpeta /wp-content/plugins/ y actívalo en tu panel de Control de Plugins
Después, añádelo a tu sidebar y configúralo en tu panel de Apariencia -> Widgets

In english:
Download the plug-in and upload the files to your /wp-content/plugins directory, and then activate ir from the plug-ins control panel
After that, add it to your sidebar and configure it in Appearance -> Widgets

== Frequently Asked Questions ==

= ¿Y qué gano yo con el widget? =
Si eres usuario de ClasificadoX, tus anuncios se mostrarán en tu web y no tendrás que hacerlo tú. Si no eres usuario de ClasificadoX, tendrás una sección de anuncios clasificados con solo instalar el widget.

=¿Influye en mi SEO al tener enlaces externos? =
Tranquilo, en tu anuncio puedes habrá enlaces a tu web, así que no hay problema
